# コーディング規約/コンベンション
- コメント・通知は日本語で丁寧に記述し、副作用を伴う関数にはコメントで明示する。
- 関数型志向を優先し、共通ヘルパーを再利用する。重複を避け、処理を呼ぶだけの中間層は作らない。
- 目安: ファイルは 150 行以内、関数は 50 行以内に収めて読みやすさを確保する。
- モジュールは `local M = {}` → ローカル関数 → `M.setup` → `return M` の順でまとまっていることが多い。
- リーダーキーは `;`。キーマップや補助関数は関心ごとに小さく分割し、関連するプラグイン設定は `lua/plugins/` に置く。
